Первая проба инструментов Python
---

Замысел.
Создать массив чисел.
Выдать квадраты чисел в массиве.
Сделать реверс массива
***

Реализация.

Возведение целых чисел мз массива в квадрат осуществляется функцией `sqware_array`.

Формат вызова функции:
sqware_array(arr)
 arr - массив целых чисел

Результат:
 массив квадратов чисел
