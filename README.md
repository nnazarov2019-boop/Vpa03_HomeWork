Первая проба инструментов Python
---

Замысел.
Создать массив чисел.
Выдать квадраты чисел в массиве.
Сделать реверс массива
Посмотреть на работу с элементарными действиями над комплексными числами
***

Реализация.

Возведение целых чисел мз массива в квадрат осуществляется функцией `sqware_array`.

Формат вызова функции:
sqware_array(arr)
 arr - массив целых чисел

Результат:
 массив квадратов чисел
---

Реверс массива осуществляется функцией `reverse_array`.

Формат вызова функции:
reverse_array(arr)
 arr - массив целых чисел

Результат:
 массив, в котором числа расположены в обратном порядке
---

Работа с комплексными числами осуществляется отдельным вычислительным блоком без выделения в отдельную функцию
